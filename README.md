# Node.js REST-client interface

This is Node.js console interface communicating with REST-API also made in Node.js.

## Setup
Install
```sh
$ npm install
```
Setup symbolic link
```sh
$ sudo npm link
```
Start application
```sh
$ node server.js
```
For help instructions about commands
```sh
$ cust -h
```

Link to REST-API: https://krizorestclient.herokuapp.com/
